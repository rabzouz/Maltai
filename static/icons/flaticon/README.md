# Flaticon icons for Maltai

Maltai can replace its built-in interface icons with local SVG files downloaded from Flaticon.

Place SVG files in this folder with these names:

```txt
attach.svg
brain.svg
chat.svg
email.svg
files.svg
library.svg
logout.svg
messages.svg
microphone.svg
model.svg
notes.svg
plus.svg
research.svg
settings.svg
tasks.svg
terminal.svg
theme.svg
tools.svg
workflow.svg
```

The app keeps its default inline icons when a file is missing, so partial icon packs are safe.

License note:
- Free Flaticon icons generally require attribution according to the selected license.
- Premium/license-owned icons can be used according to your Flaticon account terms.
- Do not scrape or automate downloads from Flaticon; download/export the icons from your Flaticon account.
