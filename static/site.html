Experience

Une interface IA faite pour travailler, pas seulement tester un modele.

Maltai regroupe chat, outils, fichiers, memoire, modeles et administration dans une seule interface lisible.

01Choisis ton provider

OpenAI, OpenRouter, Ollama local ou tout endpoint compatible OpenAI.

02Active l'agent

Laisse Maltai utiliser les outils autorises selon le plan et le role utilisateur.

03Controle l'instance

Admin console, process, Git read-only, credits, Brain et modeles Ollama.

Outils natifs

Un agent qui sait agir dans ton workspace.

Browser

Navigation web, extraction de texte, liens, formulaires, clics Playwright et captures PNG.

Fichiers

Lecture, ecriture, patch et telechargement dans un workspace isole par utilisateur.

Terminal admin

Console, process longs, logs, kill, version app et inspection Git read-only.

Brain

Memoire vectorielle, recherche, souvenirs epingles et nettoyage controle.

Ollama

Liste, telechargement avec progression et suppression des modeles locaux.

Credits

Plans Basic, Premium, Admin, suivi tokens et balance pour cadrer l'usage des outils.

Cas d'usage

Pour administrer, chercher, coder et automatiser depuis le meme endroit.

Recherche webLire des pages, extraire les liens, capturer une page et produire une synthese.
IA localeInstaller des modeles Ollama depuis l'interface et basculer entre plusieurs providers.
WorkspaceLire, creer ou patcher des fichiers dans un espace isole par utilisateur.
AdministrationSurveiller la version, les process, les logs, les credits et les acces.

Securite

Des garde-fous pour ouvrir les outils sans perdre le controle.

Roles

Basic, Premium et Admin separent le chat, les outils agent et les commandes sensibles.

Sandbox

Workspace isole par utilisateur, code Python limite et protections SSRF sur les URLs internes.

Admin only

Terminal, Python admin, process et Git read-only restent reserves aux administrateurs.

Acces

Des permissions claires pour chaque utilisateur.

Basic pour discuter, Premium pour les outils agent, Admin pour administrer l'instance.

BasicChat principal et providers configures.
PremiumOutils agent, recherche, fichiers, browser, code sandbox.
AdminConsole, terminal, process, Git read-only et usage non facture.

Installer

Telecharger et deployer Maltai.

Le projet est disponible sur GitHub. Tu peux l'installer avec Git, Docker ou le deployer sur Coolify.

git clone https://github.com/rabzouz/Maltai.git
cd Maltai
python3 -m venv venv
pip install -r requirements.txt
python app.py
