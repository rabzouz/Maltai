Sécurité

Des garde-fous pour ouvrir les outils sans perdre le contrôle.

Rôles

Basic, Premium et Admin séparent le chat, les outils agent et les commandes sensibles.

Isolation

Sessions et workspace isolés par utilisateur : chaque compte ne voit que ses propres données.

Anti-SSRF

Les outils web bloquent les adresses internes, y compris au fil des redirections.

Admin only

Terminal, Python admin, process et Git read-only restent réservés aux administrateurs.

FAQ

Questions fréquentes.

Qu'est-ce que MaltaiAI ?

MaltaiAI est un workspace IA open source que tu héberges toi-même. Il réunit le chat multi-providers, des agents capables d'utiliser des outils (web, fichiers, code, terminal), une mémoire vectorielle et une console d'administration, dans une seule interface web responsive.

Quels modèles et providers sont supportés ?

Tout endpoint compatible OpenAI : OpenAI, OpenRouter, ainsi qu'Ollama, vLLM ou llama.cpp en local. Tu peux configurer plusieurs providers et basculer de l'un à l'autre selon le besoin.

Est-ce vraiment auto-hébergé ? Où vont mes données ?

Oui. Maltai tourne sur ton serveur (Docker, Coolify, Linux, macOS ou Windows) et stocke ses données localement. Tu gardes le contrôle de tes conversations, de tes clés API et de tes fichiers.

Combien ça coûte ?

Le logiciel est open source et gratuit. Les seuls coûts éventuels sont ceux des providers d'IA payants que tu choisis de brancher (OpenAI, OpenRouter…). Avec Ollama en local, tu peux fonctionner sans coût d'API.

Comment l'installer ?

Clone le dépôt GitHub et lance l'application avec Python, ou déploie l'image Docker. Le projet est prêt pour un déploiement en un clic via Coolify. Les commandes sont détaillées dans la section Installer ci-dessous.

Est-ce sécurisé pour plusieurs utilisateurs ?

Oui. Les sessions et le workspace sont isolés par utilisateur, les rôles séparent le chat des commandes sensibles, et les outils web bloquent les adresses internes (protection anti-SSRF, même au fil des redirections).
